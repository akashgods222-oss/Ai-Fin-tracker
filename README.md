# 💸 FinTrack AI (v1.0)
### Smart Expense & Budget Analyser

FinTrack AI ek lightweight aur interactive web-based dynamic dashboard hai, jo users ko unke daily expenses track karne aur budget limits ko monitor karne me madad karta hai. Yeh project completely pure HTML, CSS aur JavaScript (DOM Manipulation) ka use karke banaya gaya hai.

---

## 🚀 Key Features

* **Real-time Pocket Analysis:** User jaise hi expense name aur amount daal kar 'Analyze Budget' par click karta hai, AI bot instantly response deta hai.
* **Smart Alert Engine:** Khache ke amount ke basis par dynamically status aur colors change hote hain:
  - ₹2000 se kam -> ✅ Safe Mode (Green)
  - ₹2000 se ₹5000 -> ⚠️ Warning Mode (Yellow)
  - ₹5000 se upar -> 🚨 Danger Mode (Red)
* **Form Auto-Reset:** Data process hone ke baad input fields automatically clear ho jati hain taaki user agla entry smoothly kar sake.
* **Premium Dark UI:** Mobile-first design ke sath ek modern aur eye-friendly dark theme layout.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic structure aur layout design ke liye.
* **CSS3:** Custom properties, modern layout styling aur smooth dynamic color transitions ke liye.
* **JavaScript (ES6):** DOM Manipulation, Event Listeners, Logic pipelines aur Reusable functional architecture ke liye.
